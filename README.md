**Answer to the questions below** 

 1. Difference between var, let, const

	•	var → old JavaScript variable, can be redeclared and function scoped.

	•	let → block scoped variable, can be reassigned but not redeclared in same scope.

	•	const → block scoped and cannot be reassigned after declaration.


2. Spread Operator (…)

    The spread operator ... is used to expand arrays or objects into individual elements.

3. map vs filter vs forEach

    map(): creates a new array after modifying each element

    filter(): returns elements that match a condition

    forEach(): loops through array but returns nothing

4. Arrow Function

    Arrow functions are a shorter syntax for writing functions.

5.  Template Literals

    Template literals use backticks (`) to create strings and insert variables.
